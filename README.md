# Sympto-Bot  – AI Voice-Enabled Health Assistant

A modern, AI-inspired voice-enabled health assistant built using **HTML, CSS, and JavaScript**. Sympto-Bot Pro allows users to describe symptoms through text or voice, receive educational information about diseases, and explore possible health conditions in a beautiful, responsive interface.

> **Educational Purpose Only**
> This project is intended for educational and informational use. It does **not** provide medical diagnosis or replace professional healthcare advice.

---

## Features

* Voice Input using the Web Speech Recognition API
* Voice Responses using the SpeechSynthesis API
* Intelligent Symptom Checker
* Built-in Medical Knowledge Base (150+ diseases)
* Disease Search and Information
* Emergency Symptom Detection
* Modern Glassmorphism User Interface
* Interactive Three.js Animated Background
* Fully Responsive Design
* Mobile and Desktop Support
* Fast Client-Side Processing
* No Backend Required
* No Database Required
* Single HTML File Application

## How It Works

1. The user enters symptoms using text or voice.
2. The application analyzes the entered symptoms.
3. The symptom checker compares them with the built-in disease database.
4. Matching diseases and educational information are displayed.
5. Emergency symptoms trigger immediate medical warnings.
6. Voice responses can read the information aloud.
